# 🚀 Nebula Notes

A lightweight, fast, and minimal note-taking web app built for developers and productivity enthusiasts.  
Nebula Notes helps you capture ideas instantly, organize them efficiently, and access them anywhere.

---

## ✨ Features

- ⚡ Blazing fast note creation
- 🗂️ Tag-based organization
- 🔍 Full-text search
- 🌙 Dark mode support
- ☁️ Optional cloud sync
- 📱 Mobile-friendly interface
